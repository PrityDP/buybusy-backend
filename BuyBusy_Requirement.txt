**Objective:
BuyBusy is a web application for the costomers of an e-commerce business.

**Goal:
Refactor the E-Commerce site to use Redux Toolkit for global state management and centralized store
to persist the state.

**Hosted Link: Solution
https://tangerine-gingersnap-3c8b63.netlify.app/

**Acceptance Criteria:
1. The app must implement global state management using Redux Toolkit.
2. The code quality of React app must be of a high standard, with proper documentation of code and 
   functionality.
3. There must be no cheating or copy-pasting of code from other sources, and all code must be oroginal
   and developed specifically for this project.

**Tasks:
1. Integrate Redux Toolkit for global state management and create centralized store to persist the state.
2. Create the required reducers and actions for mutating the state.
3. Create a login page that allows existing users to login into your app.
4. Create asynchronous thunks for performing asynchronous logic(Login/Signup, Adding/Removing product in 
    cart etc.)
5. Utilize extraReducers() to update the state based on the promise returned by asynchronous thunks, which
   include Pending, Fullfilled.
6. Update the component of E-Commerce app to use the new Redux Toolkit state management system.
7. Additionally, you can add the following functionality to the project to maximize your score:
     1. Create a Sidebar to filter the products based on product price and categories on the HomePage.
        The search and filter should be applied to the products simultaneously.
     2. Add and use functions like createSelector() and createEntityAdaptor() for better performance.
        createSelector() creates memorized selectors, which can cache the result of a calculation and
        only recompute it if the input data changes. createEntityAdaptor() creates normalized state structure
        for storing collections of entities.

** Hints:
